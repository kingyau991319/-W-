# Electron template

Minimal Electron project scaffold.

Quick start

```bash
# install deps
npm install

# start the app
npm start
```

Notes

- This template uses a devDependency on `electron`. Use `npm install` before `npm start`.
- Main entry: `src/main.js`. Preload: `src/preload.js`. Renderer: `src/index.html` + `src/renderer.js`.
